#### [[50_Filipenses|Filipenses - Resumo por Eventos]]

[[CAPITULO_1_Viver_e_Cristo]]
___
1. **Introdução ( 1:1–11 )**
    a. Saudações de Paulo e Timóteo ( 1:1–2 )
    b. Ação de Graças e Oração ( 1:3–11 )
    
2. **Prisão de Paulo ( 1:12–26 )**
    a. As provações promovem o Evangelho (  1:12⁠–⁠20  )
    b. Viver é Cristo (  1:21⁠–⁠26  )

3. **Seguindo o exemplo de Jesus (  1:27⁠–⁠2:18  )**
    a. Digno do Evangelho ( 1:27–30 )

[[CAPITULO_2_A_Mente_de_Cristo]]
___
3. Continuação:
    b. Um em Cristo ( 2:1–4 )
    c. A Mente de Cristo ( 2:5–11 )
    d. Luzes do Mundo ( 2:12–18 )

4. **Dois exemplos a seguir ( 2:19–30 )**
    a. Timóteo ( 2:19–24 )
    b. Epafrodito ( 2:25–30 )

[[CAPITULO_3_Avancando_em_direcao_ao_Gol]]
___
5. **Advertências contra falsos exemplos ( 3:1–21 )**
    a. Justiça pela fé em Cristo ( 3:1–11 )
    b. Prosseguindo em direção ao objetivo ( 3:12–16 )
    c. Cidadania no Céu ( 3:17–21 )

[[CAPITULO_4_Alegra-te_no_Senhor]]
___
6. **Desafio para Viver o Exemplo ( 4:1–23 )**
    a. Alegrai-vos no Senhor ( 4:1–9 )
    b. A Generosidade dos Filipenses ( 4:10–20 )
    c. Saudações Finais ( 4:21–23 )
